<task type="auto">
  <name>Task 2: Pixel-art consistency audit</name>
  <files>apps/web/src/index.css, apps/web/tailwind.config.ts, apps/web/src/components/ (touch-ups)</files>
  <action>
    Run through this explicit checklist; for each item, fix divergences.
    Each fix is a small commit so the audit trail is reviewable.

    **Border weight & pixel-press affordance**
    - [ ] Every primary CTA button uses the same `pixel-border` weight (4px).
          Audit: PredictTab Enter Result, LeagueTab Regenerate / Save,
          RankingTab Share, Snapshot Join CTA, Onboarding Continue, AdminLogin.
    - [ ] All clickable cards (PredictTab match cards, snapshot preview)
          consistently use `pixel-border-sm` (2px). No 1px borders anywhere.
    - [ ] The `pixel-press` interactive state (translate-y on active) is
          applied to all primary CTAs.

    **Accent color usage**
    - [ ] `text-pixel-gold` / `bg-pixel-gold` → reserved for current-user
          / featured / "you" highlight.
    - [ ] `text-pixel-green` / `bg-pixel-green` → success / saved / answer
          confirmed.
    - [ ] `text-pixel-red` / `bg-pixel-red` → danger / destructive (remove
          member, regenerate code, error toasts).
    - [ ] `text-pixel-blue` / `bg-pixel-blue` → admin/share/info actions.
    - [ ] No instance of inline hex colors in TSX (`#xxxxxx`) outside the
          deliberate exceptions: `SnapshotCard.tsx` capture mode, OG/PNG
          assets. Grep: `grep -rn "#[0-9a-fA-F]\{6\}" apps/web/src/components/`.

    **Retro typography vocabulary**
    - [ ] `tracking-[0.3em]` uppercase appears only on: status badges
          (OPEN/SAVED/LOCKED/RESULT?), the bonus reveal lock, the snapshot
          card header. Not on body copy.
    - [ ] All top-level page headers use the emoji prefix pattern:
          `🏆 Leaderboard`, `⚽ {stage}`, `👥 {league}`, `📋 Results`.

    **Layout robustness (FR rendered — verify after 05-03)**
    - [ ] No truncation/overflow visible when language = FR on:
          PredictTab match cards, LeagueTab member rows, Ranking share button,
          BonusPrediction reveal placeholder.
    - [ ] Where truncation is unavoidable, `truncate` class is applied and
          the parent has `min-w-0`.

    After fixing items, capture the final checklist state in
    `05-04-SUMMARY.md` as a `[x]` table.
  </action>
  <verify>
