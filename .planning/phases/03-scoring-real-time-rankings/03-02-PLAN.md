<task type="auto" tdd="true">
  <name>Task 1: Implement Scoring Service Logic</name>
  <files>apps/api/src/services/scoring.service.ts</files>
  <behavior>
    - Correct score (2-1 predicted, 2-1 result) -> 3 points + exact match.
    - Correct outcome (2-1 predicted, 1-0 result) -> 1 point.
    - Incorrect outcome -> 0 points.
    - Correct bonus prediction -> +2 points.
    - Leaderboard recalc -> Ranks users by Total -> Bonus -> Exact Scores.
  </behavior>
  <action>
    Implement `ScoringService`:
    1. `scoreMatch(matchId)`:
       - Fetch match result and all predictions for that match.
       - Calculate points for each prediction (3/1 + Bonus).
       - Update `predictions` table (`points`, `points_match`, `points_bonus`, `is_scored`).
       - Mark match as `is_final` if result is definitive.
    2. `recalculateLeaderboard(leagueId)`:
       - Aggregates all `is_scored` predictions for the league.
       - Calculates total points, total bonus points, and total exact matches per user.
       - Calculates new ranks with tie-breaker logic (D-05).
       - Compares with previous `leaderboard_snapshots` to derive `rank_delta`.
       - Inserts/Updates `leaderboard_snapshots`.
  </action>
  <verify>
    <automated>cd apps/api && bun test src/services/scoring.test.ts</automated>
  </verify>
  <done>Scoring logic implemented and handles all point rules and tie-breakers.</done>
